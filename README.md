# Planeaci-n_Proyecto_JuanDavidSanmiguelArtunduaga

Propósito del proyecto

El propósito del proyecto es diseñar y desarrollar un sistema de software que permita mejorar y optimizar el proceso de gestión de compras en línea, brindando una herramienta eficiente, segura y fácil de usar tanto para los clientes como para los administradores. Con este proyecto se busca solucionar problemas actuales como la desorganización en pedidos, retrasos en la gestión y falta de control de inventarios.



Fases a desarrollar

Análisis de requerimientos: Recolección y documentación de las necesidades del usuario y del sistema.

Diseño: Elaboración de diagramas, estructura del sistema, base de datos e interfaz.

Desarrollo: Programación de las funcionalidades del sistema según los requerimientos definidos.

Pruebas: Validación del funcionamiento del software, corrección de errores y ajustes necesarios.

Implementación: Puesta en marcha del sistema para su uso real.

Mantenimiento: Mejoras continuas, soporte técnico y actualización del sistema según las necesidades del usuario.
